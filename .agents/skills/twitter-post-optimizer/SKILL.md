# Write X/Twitter Copy

Turn a post or idea into concise, specific social copy without making it sound promotional.

## Ground the Draft

1. Read the supplied draft or source post completely enough to identify its actual conclusion.
2. Extract the strongest concrete detail: a result, distinction, mechanism, limitation, personal observation, or useful question.
3. When matching prior social voice, search recent post Markdown for embedded `twitter-tweet` blocks and inspect the surrounding argument. Do not infer a platform-wide strategy from one example.
4. If the post is unpublished, use a link placeholder unless the user provides the final URL.

## Write

- Put the main idea in the first sentence; omit throat-clearing.
- Prefer a specific claim or question over a generic announcement such as “I wrote a new post.”
- Preserve technical vocabulary when it helps the intended readers.
- Preserve limitations and uncertainty. Do not turn a qualified finding into a universal claim.
- Use first person when the source contains a real observation, choice, or reaction.
- Keep calls to action low pressure; the content should carry the post.
- Use emojis and hashtags sparingly and only when they suit the supplied draft or requested format.
- Adapt Japanese and English naturally rather than translating word for word.
- Do not default to bilingual output, a thread, or multiple variants unless the user asks or would clearly benefit.

## Threads

- Give each post one job and one new piece of information.
- Make the opener understandable without promising more than the thread delivers.
- Order posts by argument, not by arbitrary list length.
- End when the explanation is complete; do not pad to a target count.

## Check

- Verify every factual claim against the source.
- Remove clickbait, inflated certainty, and generic engagement prompts.
- Ensure the hook and linked article make the same promise.
- Count characters with the actual URL or placeholder when a hard platform limit matters; verify current platform rules rather than relying on a memorized limit.
