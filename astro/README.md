# Astro Scaffold

This directory contains the first Astro scaffold for the Hippocampus's Garden rebuild.

## Included

- content collections for posts
- Home page with pagination
- post route
- Topics index and topic archive
- About page scaffold
- RSS route
- shared layouts, styles, and data files

## Commands

From the repository root:

```bash
npm run astro:migrate-posts
npm run astro:dev
npm run astro:check
npm run astro:build
```

Or from this directory:

```bash
npm install
npm run dev
```
