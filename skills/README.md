# Repository Skills (Adapted)

These skills were extracted from patterns in external skill repositories and adapted to this Gatsby blog (`Hippocampus's Garden`).

## Added Skills

- `gatsby-blog-post-authoring`
  - Adapts content drafting/research collaboration workflows for `content/blog/*/index.md` and this repo's frontmatter/tag conventions.
  - Inspired by: open-source content writing skills (notably `content-research-writer`).

- `gatsby-blog-image-workflow`
  - Adapts screenshot/image enhancement and verification workflows for inline images and `featuredImage`/OGP assets in blog post folders.
  - Inspired by: screenshot/image preparation skills (`screenshot`, `image-enhancer`).

- `gatsby-blog-ui-review`
  - Adapts UI review and frontend QA/design guidance to Gatsby templates/components used by this blog.
  - Inspired by: `webapp-testing`, `frontend-design`, `web-design-guidelines`.

- `gatsby-blog-preview-publish`
  - Adapts preview/smoke-test/deploy workflows to this repo's npm scripts, Gatsby build behavior, and Netlify usage.
  - Inspired by: `netlify-deploy` plus webapp testing workflow patterns.

- `blog-writing-style-en`
  - Encodes the author's recent English blog voice (technical explainers/reviews) from 10 recent EN posts; includes voice-preserving polishing guidance.
